# Examples

Embed a whole file:

```embed
f1.rb
```

Embed multiple whole files:

```embed
f1.rb
f2.go
```

Embed specific lines in a file:

```embed
f3.js log
```

Embed multiple whole files and multiple blocks within files:

```embed
f1.rb
f2.go log
f4.css h1
f5.html h1
f5.html ul
f6.sql users
```

Embed Markdown files and their embeds recursively:

```embed
input2.md
```
